SecureTrustOTA

A Resilient and Trust-Based Over-the-Air Firmware Update Framework for IoT Devices, integrating Zero-Trust Authorization, secure delivery, fail-safe rollback, and anomaly detection.

Abstract

What is SecureTrustOTA?

SecureTrustOTA is a modular, four-layered OTA firmware security framework designed for resource-constrained IoT environments. It enforces continuous Zero Trust authorization and per-transaction device verification, evaluates device health through lightweight real-time anomaly detection, optimizes firmware delivery using an instruction-aware delta patching engine, and guarantees post-update stability through active hardware attestation and autonomous rollback all working together to secure not just the firmware package, but the entire update lifecycle.
Framework Architecture

Four Integrated Layers

SecureTrustOTA is built on four tightly integrated components, each addressing a critical dimension of IoT firmware security.

Zero-Trust Authorization Layer

Enforces continuous authentication, dynamic policy-based access control, firmware attestation, and tamper-evident audit logging. No implicit trust at any stage.

Layer #1 · Core Component

Lightweight Secure OTA Core

Ensures efficient, encrypted update transfer using ECC and AES-CCM encryption, delta updates, and transmission strategies tailored for low-power devices.

Layer #2

Threshold-Based Anomaly Detection

Continuously monitors key OTA parameters - update size, memory usage, timing — to flag abnormal or malicious update behaviour before it causes harm.

Layer #3

Resilient Fail-Safe Update Engine

Safeguards device availability through dual-slot updates, rollback prevention, secure logging, and recovery scheduling to minimise downtime in all failure scenarios.
